# 📖 КАК ОТКРЫТЬ ЛОГИ ВЫПОЛНЕНИЯ (Пошаговая инструкция)

## 🎯 ШАГ ЗА ШАГОМ:

### ШАГ 1: Найдите список запусков

Вы видите таблицу с запусками:
```
Function          Type      Start Time                    Duration   Status
Version 8  onSheetEdit  Trigger  Nov 4, 2025, 5:18:30 PM  1.348 s   Completed
```

### ШАГ 2: Нажмите на время запуска

**Нажмите на одно из времен**, например:
- `Nov 4, 2025, 5:18:30 PM` ← **КЛИКНИТЕ СЮДА**
- ИЛИ `Nov 4, 2025, 5:18:21 PM` ← ИЛИ СЮДА

### ШАГ 3: Откроется страница с деталями

Вы увидите страницу с информацией о запуске:
- **Execution type:** Trigger
- **Function:** onSheetEdit
- **Start time:** Nov 4, 2025, 5:18:30 PM
- **Duration:** 1.348 s

### ШАГ 4: Прокрутите вниз

**Прокрутите страницу ВНИЗ**, пока не увидите раздел:

**"Execution log"** или **"Логи выполнения"**

### ШАГ 5: Скопируйте логи

В разделе "Execution log" будут записи, например:
```
[2025-11-04T22:18:30.123Z] [START] === onSheetEdit triggered ===
[2025-11-04T22:18:30.125Z] [INFO] Edit detected: Row 5, Column 2...
```

**Выделите ВСЕ эти записи** и скопируйте (Ctrl+C / Cmd+C)

### ШАГ 6: Отправьте мне

Вставьте скопированные логи сюда, чтобы я мог их проанализировать.

---

## ❓ Если логи пустые или их нет:

Это означает, что:
1. Либо код триггера не обновлен (используется старая версия)
2. Либо логирование не работает

**В этом случае:**
1. Убедитесь, что вы скопировали **весь код** из `GOOGLE_APPS_SCRIPT_CODE.js`
2. Вставили его в Google Apps Script
3. Сохранили (Ctrl+S)

---

## 🔍 Альтернативный способ: Посмотреть все логи сразу

1. В Google Apps Script слева найдите **"View"** → **"Logs"** (или "Логи")
2. Там будут все логи за последнее время
3. Скопируйте логи, которые начинаются с `[START] === onSheetEdit triggered ===`
