✅ Вы уже получаете тестовые email - разрешения настроены!

Теперь нужно настроить ТРИГГЕР для автоматической отправки:

1. Откройте https://script.google.com
2. Найдите ваш проект
3. Нажмите на иконку ⏰ (Triggers) слева
4. Нажмите + Add Trigger
5. Настройки:
   Function to run: onSheetEdit
   Event source: From spreadsheet
   Event type: On edit
6. Нажмите Save
7. Предоставьте разрешения (если попросит)

ГОТОВО! Теперь email будет отправляться автоматически при каждом новом лиде.
